# My Paisa HQ

Free finance calculators for Indian professionals. Everything updates in real time, no login required.

**Live site:** [mypaisahq.com](https://mypaisahq.com)

---

## Calculators

| Calculator | What it does |
|---|---|
| **8th Pay Commission** | Revised salary based on fitment factor and allowances |
| **Gratuity Calculator** | Gratuity amount, tax exemption, and taxable portion |
| **In-Hand Salary / CTC** | CTC breakdown with old vs new tax regime comparison |
| **HRA Exemption** | HRA claim calculation by city and salary |
| **Salary Hike Calculator** | New CTC after a percentage raise |
| **Income Tax FY 2025-26** | Slab-wise tax with regime comparison |
| **SIP Returns** | SIP maturity value and growth over time |
| **Loan vs SIP** | Whether to prepay a loan or invest in SIP |
| **Goal-based SIP** | Monthly SIP needed to hit a financial target |

## Tech stack

- TypeScript
- React (client)
- Node.js (server)
- Drizzle ORM
- Vercel (deployment)
- Built on Replit

## Project structure

```
api/              Serverless API handlers
client/           React frontend
server/           Backend logic
shared/           Shared types and utilities
script/           Build scripts
attached_assets/  Static assets
```

## Running locally

```bash
npm install
npm run dev
```
